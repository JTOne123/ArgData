# SetupTyreCompound

Tyre compound in setup file. Can be A, B, C or D.



## Fields

| Name            | Description        |
|-----------------|--------------------|
| A   |  Tyre compound A, the hardest tyre. 
| B   |  Tyre compound B. 
| C   |  Tyre compound C. 
| D   |  Tyre compound D, the softest tyre (except qualifying tyres).
