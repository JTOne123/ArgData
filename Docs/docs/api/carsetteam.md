# CarSetTeam

Defines a team as it appears in a CarSet.



## Constructors

| Name            | Description        |
|-----------------|--------------------|
| CarSetTeam() |  Initializes a new CarSetTeam. 


## Properties

| Name            | Description        |
|-----------------|--------------------|
| Car   |  Car with colors. 
| Drivers   |  Drivers in the team. 
| Horsepower   |  Gets or sets the horsepower value of the team's car. 
| PitCrew   |  PitCrew with colors.
